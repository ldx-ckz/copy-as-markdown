# Copy as Markdown

Edge浏览器插件，将选中的网页内容转换为Markdown格式并复制到剪贴板

## 功能

- 将选中的HTML内容转换为Markdown格式
- 支持数学公式转换（LaTeX格式）
- 保留链接和图片
- 一键复制到剪贴板
- 支持快捷键操作

## 使用方法

1. 在网页上选择想要复制的内容
2. 使用以下任一方式复制：
   - 点击工具栏插件图标，点击"复制选中内容"
   - 使用快捷键 `Ctrl+Shift+F`

## 安装

1. 下载插件文件
2. 打开Edge浏览器，访问 `edge://extensions/`
3. 开启"开发人员模式"
4. 点击"加载解压缩的扩展"，选择插件文件夹

## 支持的网站

- Wikipedia（数学公式优化）
- 新闻和博客网站
- 技术文档
- 普通网页内容
